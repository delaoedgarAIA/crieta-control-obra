# Crieta - Sistema Integral de Control de Obra

Control inteligente de presupuestos, órdenes de cambio y seguimiento de proyectos de construcción.

## 🏗️ Características

- **Captura de datos en obra** - Carga fotos de anotaciones
- **Búsqueda inteligente** - Autocompletado tipo Google referenciado al presupuesto
- **Validación automática** - Sistema sugiere partidas presupuestadas
- **Generador de OdC** - Órdenes de cambio automáticas con cálculos integrados
- **Dashboard en tiempo real** - Presupuesto vs Real ejecutado
- **Aprendizaje continuo** - Sistema recuerda ajustes de precios

## 📊 Proyecto

**Hotel Casa de Mar - Adecuación Suites & Cocina**
- Presupuesto Total: $284,661.44
- Área: 331.34 m²
- Duración: 5 meses

## 🚀 Cómo usar

1. Abre el sitio en `crieta.app`
2. Ve a **Captura de Datos** y sube una foto
3. Analiza con IA (simula extracción de números)
4. Valida en **Validación** contra partidas presupuestadas
5. Genera órdenes de cambio en **Órdenes de Cambio**
6. Visualiza seguimiento en **Dashboard**

## 🔧 Tecnología

- HTML5 + CSS3 + JavaScript vanilla
- Responde en móvil y desktop
- Sin dependencias externas
- Almacenamiento local del navegador
